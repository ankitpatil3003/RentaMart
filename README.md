# RentaMart

US rental marketplace (hybrid renter + landlord). Layer 1 in progress: search, apply, application fee.

## Stack

Next.js, Convex, Clerk, Stripe (test mode).

## Quick start

Setup details will expand as Layer 1 lands. For now:

```bash
npm install
npm run dev
```

Do not commit secrets. Use `.env.example` once it is added.
